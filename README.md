# AI Chatbot with Natural Language Processing

This project is a simple AI-powered chatbot built using Python and Natural Language Processing (NLP) techniques with the NLTK library. 
The chatbot is designed to understand basic user inputs and respond appropriately using predefined intents and patterns. 
It demonstrates how text preprocessing, tokenization, and stemming can be used to build conversational systems.

## Features

Responds to greetings, farewells, thanks, and general queries

Uses NLTK tokenization and stemming for text processing

Matches user input with predefined intent patterns

Provides meaningful responses in real time

Runs directly in the terminal or command prompt
Installation

## Clone the repository:

git clone <repository_link>


## Install required libraries:

pip install nltk


## Download NLTK tokenizer data:

import nltk
nltk.download('punkt')

## Usage

Run the chatbot script:

python chatbot.py

Type a message and press Enter.

Type quit to exit the chat.
